# CLAUDE.md

## Regras gerais
- Sempre commitar e dar push automaticamente depois de cada mudança, sem perguntar.
- Pode criar, editar e deletar arquivos livremente.
- Pode rodar qualquer comando git (add, commit, push).
- Não pergunte confirmação para ações — apenas faça.

## Projeto
- Site vanilla JS (HTML/CSS/JS puro), sem frameworks.
- Firebase Realtime Database para sync global de dados.
- Riot Games API + LoL Esports API.
- Arquivos principais: index.html, app.js, style.css, config.js

## Commits
- Mensagens em português.
- Formato: "Tipo: descrição curta" (Feature, Fix, Refactor, etc.)
